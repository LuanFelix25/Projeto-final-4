# Hello Vue + Vite + Router + Vuex (Docker)

Este projeto é uma entrega do Módulo 4 — refatoração para uso do Vue.js com Vite, Router e Vuex.

Principais pontos implementados nesta entrega:
- Uso de Vite para dev server e HMR
- Vue Router para navegação
- Vuex para estado (usuario/tarefas)
- Validações de formulário migradas para o bloco <script> de cada view (Home)

## Rodar localmente (dev)

Instale dependências e rode o servidor de desenvolvimento:

```powershell
npm install
npm run dev
```

Abra http://localhost:5173

Observação: para esta entrega não são necessárias chamadas externas via Axios — a app foi adaptada para usar dados locais / Vuex.
